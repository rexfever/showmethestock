# 수동 매매 가이드

## 📊 수동 매매 개요

자동매매가 아닌 수동으로 매매하는 경우를 위한 가이드입니다. 추천 종목을 직접 선택하여 매수하고, 수익률을 모니터링하며 매도하는 방법을 안내합니다.

---

## 🎯 1단계: 추천 종목 확인

### 추천가 확인 방법

1. **스캐너 페이지 접속**
   - 추천 종목 목록 확인
   - 각 종목의 **추천가(추천일 종가)** 확인
   - **현재가(오늘 종가)** 확인

2. **추천가 vs 현재가**
   - **추천가**: 스캔일 종가 (매수 기준가)
   - **현재가**: 오늘 종가 (현재 시점의 가격)
   - 현재가가 추천가보다 낮으면 매수 기회
   - 현재가가 추천가보다 높으면 이미 상승한 상태

3. **점수 확인**
   - **8점 이상**: 높은 성공률 기대, 적극 매수 권장
   - **10점 이상**: 최고 성과 기대, 우선 매수 권장

---

## 💰 2단계: 매수 타이밍

### 매수 가격 기준

**권장: 추천가 기준으로 매수**

- **이상적**: 추천가 근처에서 매수
- **허용 범위**: 추천가의 ±2% 이내
- **주의**: 추천가보다 5% 이상 높으면 매수 지양

### 매수 타이밍 전략

#### 시나리오 1: 추천일 당일 매수
- **조건**: 스캔 결과 확인 후 당일 장중 매수
- **가격**: 추천가(전일 종가) 근처에서 매수
- **장점**: 추천 타이밍과 가장 가까움
- **주의**: 시초가 확인 후 매수 권장

#### 시나리오 2: 추천일 다음날 매수
- **조건**: 추천일 다음날 장중 매수
- **가격**: 추천가 근처 또는 그보다 낮은 가격에서 매수
- **장점**: 가격 확인 후 여유롭게 매수 가능
- **주의**: 이미 상승한 경우 매수 지양

#### 시나리오 3: 하락 시 매수
- **조건**: 추천가보다 2~5% 하락한 경우
- **가격**: 하락한 가격에서 매수
- **장점**: 더 낮은 가격에 매수 가능
- **주의**: 과도한 하락(-10% 이상)은 피하기

### 매수 수량 결정

**종목당 투자금액**
- 권장: 종목당 100만원 ~ 200만원
- 예시: 10,000원 주식 → 10주 ~ 20주 매수

**포트폴리오 분산**
- 동시 보유 종목: 3~5개 권장
- 투자 자금 1,000만원 기준: 종목당 200만원씩

---

## 📈 3단계: 포트폴리오 등록

### 투자 등록 방법

1. **스캐너 페이지에서 등록**
   - 추천 종목 카드에서 "투자 등록" 버튼 클릭
   - 매수가(추천가 또는 실제 매수가), 수량, 매수일 입력
   - 등록 완료

2. **포트폴리오 페이지에서 등록**
   - 포트폴리오 페이지 접속
   - "종목 추가" 버튼 클릭
   - 종목 정보 입력 후 등록

### 등록 정보
- **종목 코드**: 티커 (예: 005930)
- **종목명**: 종목 이름
- **매수가**: 실제 매수한 가격
- **수량**: 매수한 주식 수
- **매수일**: 매수한 날짜

---

## 📊 4단계: 수익률 모니터링

### 수익률 확인 방법

1. **포트폴리오 페이지**
   - 보유 종목 목록 확인
   - 현재 수익률 실시간 확인
   - 최대 수익률/최소 수익률 확인

2. **수익률 계산**
   - **현재 수익률**: (현재가 - 매수가) / 매수가 × 100
   - **보유 기간**: 매수일로부터 경과 일수

### 모니터링 주기

- **매일 1회**: 장 마감 후 수익률 확인
- **중요 시점**: 익절/손절 기준 근처에서 자주 확인
- **보유 기간**: 최소 5일, 최대 45일

---

## 🎯 5단계: 매도 타이밍

### 익절: +3% 도달 시 매도

**조건**
- 현재가가 매수가의 103% 이상 도달
- 예: 10,000원 매수 → 10,300원 도달 시 매도

**실행 방법**
1. 포트폴리오에서 수익률 확인
2. +3% 이상 도달 시 즉시 매도
3. 매도 내역을 포트폴리오에 기록

**주의사항**
- 감정에 휘둘리지 말고 기준대로 실행
- +3% 도달 후 추가 상승 기대하지 말기
- 빠른 수익 실현이 목표

### 손절: -7% 하락 시 매도 (5일 후)

**조건**
- 매수 후 5일 경과
- 현재가가 매수가의 93% 이하로 하락
- 예: 10,000원 매수 → 5일 후 9,300원 이하 시 매도

**실행 방법**
1. 매수 후 5일은 기다리기 (조정 가능)
2. 5일 후에도 -7% 이상 하락 시 매도
3. 매도 내역을 포트폴리오에 기록

**주의사항**
- 매수 후 5일 이내에는 매도하지 않기 (조정 가능)
- 손실 확대 방지가 목표
- 감정적 판단보다 기준 준수

### 보존: 작은 수익도 지키기

**조건**
1. 현재가가 매수가의 101.5% 도달 (1.5% 수익)
2. 이후 원가(매수가) 이하로 하락 시 매도
3. 예: 10,000원 매수 → 10,150원 도달 → 다시 10,000원 이하 시 매도

**실행 방법**
1. +1.5% 도달 시 보존 모드 활성화
2. 이후 원가 이하로 떨어지면 즉시 매도
3. 손실 없이 수익 보존

**주의사항**
- 작은 수익이라도 지키는 것이 중요
- 원가 이하로 떨어지면 즉시 매도
- 추가 상승 기대하지 말기

### 보유 기간 관리

**최소 보유 기간: 5일**
- 매수 후 최소 5일은 보유
- 너무 빨리 판매하지 않기
- 승률 향상을 위해 필요

**최대 보유 기간: 45일**
- 45일 넘게 보유하지 않기
- 보통 30일 안에 매매 완료
- 장기 보유는 비효율적

---

## 📝 6단계: 매매 내역 기록

### 매매 내역 기록 방법

1. **포트폴리오 페이지**
   - 보유 종목에서 "매매 내역 추가" 버튼 클릭
   - 매수/매도 선택
   - 수량, 가격, 날짜 입력
   - 기록 완료

2. **기록 정보**
   - **매수**: 매수가, 수량, 매수일
   - **매도**: 매도가, 수량, 매도일
   - **비고**: 특이사항 기록 (선택)

### 기록의 중요성

- **수익률 추적**: 정확한 수익률 계산
- **전략 개선**: 매매 패턴 분석
- **세금 신고**: 필요 시 자료 활용

## 💡 실전 팁

### 종목 선택

**우선순위**
1. **10점 이상**: 최우선 매수
2. **8점 이상**: 적극 매수
3. **6~7점**: 신중히 검토 후 매수

**제외할 종목**
- 점수가 너무 낮은 종목 (5점 이하)
- 이미 크게 상승한 종목 (추천가 대비 +10% 이상)
- 거래량이 너무 적은 종목

### 시장 상황별 전략

**시장이 좋을 때 (상승장)**
- 더 적극적으로 매수
- 익절 목표를 4%까지 올려도 됨
- 추천 종목이 많을 수 있음

**시장이 안 좋을 때 (하락장)**
- 조심스럽게 접근
- 추천 종목이 적을 수 있음
- 손절 기준을 엄격히 준수

### 감정 관리

**하지 말아야 할 것**
- ❌ FOMO (놓칠까봐 두려워서 급하게 매수)
- ❌ 손실 확대 (손절 기준 무시하고 보유)
- ❌ 수익 과도 기대 (+3% 도달 후 추가 보유)
- ❌ 감정적 판단 (기준 무시하고 임의 판단)

**해야 할 것**
- ✅ 기준 준수 (익절/손절 기준 엄격히 준수)
- ✅ 냉정한 판단 (감정 배제하고 객관적 판단)
- ✅ 빠른 실행 (익절/손절 기준 도달 시 즉시 실행)
- ✅ 기록 관리 (모든 매매 내역 정확히 기록)

- **승률**: 94.6%

- **손절률**: 2.4%
- **평균 수익률**: +9.97%
- **평균 보유 기간**: 32.9일

### 주요 특징
- 익절률 93.4%로 매우 높은 성공률
- 손절률 2.4%로 낮은 손실 확률
- 평균 수익률 +9.97%로 우수한 성과

## 💰 실전 예시

### 시나리오 1: 성공 케이스 (수동 매매)
1. **추천 확인**: 2025-01-15 스캔에서 A종목 추천 (추천가: 10,000원, 점수: 9점)
2. **매수**: 2025-01-16 장중 10,000원에 10주 매수 (10만원 투자)
3. **포트폴리오 등록**: 매수가 10,000원, 수량 10주, 매수일 2025-01-16 등록
4. **모니터링**: 매일 수익률 확인
5. **익절**: 2025-01-19 현재가 10,300원 도달 → 즉시 매도
6. **수익**: 3,000원 수익 실현 (+3%)

### 시나리오 2: 보존 케이스 (수동 매매)
1. **매수**: 2025-01-15 10,000원에 10주 매수
2. **포트폴리오 등록**: 매수가 10,000원, 수량 10주 등록
3. **상승**: 2025-01-17 현재가 10,150원 도달 (+1.5%)
4. **보존 모드**: 원가 이하로 떨어지면 매도 준비
5. **하락**: 2025-01-20 현재가 9,900원으로 하락 → 즉시 매도
6. **결과**: 손실 없음 (보존 성공)

### 시나리오 3: 손절 케이스 (수동 매매)
1. **매수**: 2025-01-15 10,000원에 10주 매수
2. **포트폴리오 등록**: 매수가 10,000원, 수량 10주 등록
3. **하락**: 매수 후 계속 하락
4. **5일 경과**: 2025-01-20 (5일 후)
5. **손절**: 2025-01-21 현재가 9,200원으로 하락 (-8%) → 즉시 매도
6. **손실**: 8,000원 손실 (-8%, 손절 기준 -7% 초과)

## ⚠️ 주의사항

### 매수 시 주의사항
- 추천가보다 5% 이상 높으면 매수하지 않기
- 거래량이 너무 적은 종목은 피하기
- 점수가 너무 낮은 종목(5점 이하)은 피하기

### 매도 시 주의사항
- 익절 기준(+3%) 도달 시 즉시 매도 (추가 상승 기대하지 말기)
- 손절 기준(-7%) 도달 시 즉시 매도 (손실 확대 방지)
- 보유 기간(5~45일) 준수

### 포트폴리오 관리 주의사항
- 동시에 3~5개 종목만 보유
- 모든 매매 내역 정확히 기록
- 수익률을 정기적으로 확인

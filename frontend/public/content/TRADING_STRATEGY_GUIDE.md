## 🎯 핵심 전략

### 1. 익절: +3% 도달 시 매도

**키움증권 자동감시매도 설정:**
- 자동감시매도 설정 방법은 [키움증권 자동감시주문 사용하기](https://youtu.be/_UHLQQ_JF-M?si=wcCeeHU4XjLeVESg) 링크를 참고하세요.
- 익절 조건 설정:
  - 감시가격: 매수가의 103% (10,000원 매수 시 10,300원)
  - 감시조건: 현재가가 감시가격 이상 도달 시
  - 주문: 시장가 매도

**어떻게 하나요?**
- 매수한 가격의 3% 오르면 자동으로 매도됨
- 예: 10,000원에 매수 → 10,300원 도달 시 자동 매도

**왜 하나요?**
- 검증 결과 승률 94.6% 확인 (개선된 스캔 로직)
- 익절률 93.4%로 높은 성공률
- 안정적인 수익 실현
- 자동으로 처리되어 놓치지 않음

### 2. 손절: -7% 하락 시 매도 (5일 후)

**키움증권 자동감시매도 설정:**
- 자동감시매도 설정 방법은 [키움증권 자동감시주문 사용하기](https://youtu.be/_UHLQQ_JF-M?si=wcCeeHU4XjLeVESg) 링크를 참고하세요.
- 손절 조건 설정:
  - 감시가격: 매수가의 93% (10,000원 매수 시 9,300원)
  - 감시조건: 현재가가 감시가격 이하로 떨어질 시
  - 주문: 시장가 매도
  - 유효기간: 매수일로부터 5일 후부터 적용

**어떻게 하나요?**
- 매수 후 5일은 기다리기 (자동감시는 5일 후 활성화)
- 5일 후에도 7% 이상 떨어지면 자동으로 매도됨
- 예: 10,000원에 매수 → 5일 후 9,300원 이하 시 자동 매도

**왜 하나요?**
- 손실이 커지기 전에 자동으로 매도
- 자금을 보존해 다시 투자 가능
- 검증 결과 손절 확률 2.4%로 낮음 (개선된 스캔 로직)
- 감정에 휘둘리지 않고 자동 처리

### 3. 보존: 작은 수익도 지키기

**키움증권 자동감시매도 설정:**
- 자동감시매도 설정 방법은 [키움증권 자동감시주문 사용하기](https://youtu.be/_UHLQQ_JF-M?si=wcCeeHU4XjLeVESg) 링크를 참고하세요.
- 보존 조건 설정:
  - 1단계: 매수가의 101.5% 도달 시 활성화 (10,000원 매수 시 10,150원)
  - 2단계: 활성화 후 원가 이하로 하락 시 매도
  - 주문: 시장가 매도
  - 트레일링 스탑 기능 활용 권장

**어떻게 하나요?**
1. 매수 후 1.5% 오르면 자동감시 활성화
2. 이후 원가 이하로 떨어지면 자동으로 매도
3. 예: 10,000원에 매수 → 10,150원 도달 → 다시 10,000원 이하로 떨어지면 자동 매도

**왜 하나요?**
- 작은 수익이라도 지킬 수 있음
- 손실 없이 자동 매도 가능
- 트레일링 스탑 활용 시 더 많은 수익 확보 가능

### 4. 보유 기간

**최소 5일 보유**
- 매수 후 최소 5일은 보유
- 너무 빨리 판매하지 않기
- 이렇게 하면 승률이 높아짐

**최대 45일 보유**
- 45일 넘게 보유하지 않기
- 보통 30일 안에 매매 완료됨

## 🔧 키움증권 자동감시매도 활용 가이드

### 자동감시매도 설정 방법

자동감시매도의 상세한 설정 방법은 다음 링크를 참고하세요:
- **[키움증권 자동감시주문 사용하기](https://youtu.be/_UHLQQ_JF-M?si=wcCeeHU4XjLeVESg)**

### 자동감시매도의 장점

**편리함**
- 하루 종일 모니터링할 필요 없음
- 설정만 해두면 자동으로 처리됨
- 감정에 휘둘리지 않고 철저히 실행

**정확성**
- 사람이 놓칠 수 있는 타이밍도 정확히 포착
- 빠른 반응으로 최적의 가격에 매도 가능

### 설정 시 주의사항

**1. 감시가격 확인**
- 매수 후 즉시 정확한 감시가격 설정
- 계산 실수 방지를 위해 계산기 활용 권장

**2. 유효기간 확인**
- 손절은 5일 후부터 적용되도록 설정
- 익절은 즉시 적용 가능
- 감시기간은 최대 30일까지 설정 가능

**3. 중복 설정 방지**
- 같은 종목에 여러 개의 감시매도가 설정되지 않도록 주의
- 기존 감시매도 현황을 먼저 확인

**4. 시장가 vs 지정가**
- 익절/손절은 시장가 매도 권장 (빠른 체결)
- 보존 전략은 지정가도 고려 가능

## 💡 실전 팁

### 종목 수 관리

**권장**: 동시에 3~5개 종목만 보유

**예시**:
- 투자 자금 1,000만원 있으면
- 종목당 200만원씩 투자 (최대 5개)

### 점수 활용

**8점 이상 종목**
- 검증 결과 높은 성공률 기대
- 개선된 스캔 로직으로 품질 향상
- 적극적으로 매수해도 좋음

**10점 이상 종목**
- 가장 좋은 성과 기대
- 우선적으로 매수 권장

### 시장 상황

**시장이 좋을 때**
- 더 적극적으로 매수
- 익절 목표를 4%까지 올려도 됨

**시장이 안 좋을 때**
- 조심스럽게 접근
- 추천 종목이 적을 수 있음

## 💰 실전 예시

### 시나리오 1: 성공 케이스 (자동감시매도 활용)
1. 10,000원에 매수
2. 키움 자동감시매도 설정: 익절 10,300원
3. 3일 후 10,300원 도달 → 자동 매도 실행
4. 수익 300원 실현

### 시나리오 2: 보존 케이스 (자동감시매도 활용)
1. 10,000원에 매수
2. 키움 자동감시매도 설정: 보존 10,150원 도달 후 원가 이하 시 매도
3. 2일 후 10,150원 도달 → 보존 모드 활성화
4. 5일 후 9,900원으로 하락 → 자동 매도 실행
5. 손실 없음 (보존 성공)

### 시나리오 3: 손절 케이스 (자동감시매도 활용)
1. 10,000원에 매수
2. 키움 자동감시매도 설정: 손절 9,300원 (5일 후부터 유효)
3. 5일 후에도 하락
4. 9,300원 이하로 떨어짐 → 자동 매도 실행
5. 손실 700원 (확대 방지)

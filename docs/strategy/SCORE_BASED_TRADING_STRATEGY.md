# 점수 기반 매매 전략 분류

## 개요

점수에 따라 매매 전략을 **스윙**, **포지션**, **장기**로 구분하여 각 전략에 맞는 매매 방식을 제안합니다.

## 매매 전략 분류

### 1. 스윙 트레이딩 (단기)

**특징:**
- 보유 기간: 수일 ~ 수주 (3~10일)
- 목표 수익률: 3~5%
- 손절 기준: -3~-5%
- 빠른 진입/청산

**점수 기준:**
- **10점 이상**: 강한 단기 모멘텀
- 빠른 수익 실현 가능
- 높은 거래량 + 강한 기술적 신호

**적합한 종목:**
- 골든크로스 + 거래량 급증 + 강한 모멘텀
- 단기 상승 추세가 명확한 종목

### 2. 포지션 트레이딩 (중기)

**특징:**
- 보유 기간: 수주 ~ 수개월 (2주~3개월)
- 목표 수익률: 5~10%
- 손절 기준: -5~-7%
- 추세 추종 매매

**점수 기준:**
- **8~9점**: 중기 추세 형성
- 안정적인 상승 추세
- 지속 가능한 모멘텀

**적합한 종목:**
- 골든크로스 + 추세 지표 + 자금 유입
- 중기 상승 추세가 형성된 종목

### 3. 장기 투자 (장기)

**특징:**
- 보유 기간: 수개월 ~ 수년 (3개월 이상)
- 목표 수익률: 10% 이상
- 손절 기준: -10% 이상
- 가치 투자 + 추세 추종

**점수 기준:**
- **6~7점**: 장기 추세 형성 가능
- 안정적인 기업 + 기술적 신호
- 지속 가능한 성장 가능성

**적합한 종목:**
- 기본 신호 충족 + 안정적인 추세
- 장기 성장 가능성이 있는 종목

## 점수별 매매 전략 매핑

| 점수 범위 | 레이블 | 매매 전략 | 보유 기간 | 목표 수익률 | 손절 기준 |
|----------|--------|----------|----------|------------|----------|
| **10점 이상** | "강한 매수" | **스윙** | 3~10일 | 3~5% | -3~-5% |
| **8~9점** | "매수 후보" | **포지션** | 2주~3개월 | 5~10% | -5~-7% |
| **6~7점** | "관심 종목" | **장기** | 3개월 이상 | 10% 이상 | -10% 이상 |
| **6점 미만** | "후보 종목" | **관찰** | - | - | - |

## 구현 방안

### 1. 레이블 확장

**현재:**
```python
if adjusted_score >= 10:
    flags["label"] = "강한 매수"
elif adjusted_score >= 8:
    flags["label"] = "매수 후보"
elif adjusted_score >= 6:
    flags["label"] = "관심 종목"
else:
    flags["label"] = "후보 종목"
```

**개선안:**
```python
if adjusted_score >= 10:
    flags["label"] = "강한 매수"
    flags["trading_strategy"] = "스윙"
    flags["target_profit"] = 0.05  # 5%
    flags["stop_loss"] = -0.05     # -5%
    flags["holding_period"] = "3~10일"
elif adjusted_score >= 8:
    flags["label"] = "매수 후보"
    flags["trading_strategy"] = "포지션"
    flags["target_profit"] = 0.10  # 10%
    flags["stop_loss"] = -0.07     # -7%
    flags["holding_period"] = "2주~3개월"
elif adjusted_score >= 6:
    flags["label"] = "관심 종목"
    flags["trading_strategy"] = "장기"
    flags["target_profit"] = 0.15  # 15%
    flags["stop_loss"] = -0.10     # -10%
    flags["holding_period"] = "3개월 이상"
else:
    flags["label"] = "후보 종목"
    flags["trading_strategy"] = "관찰"
    flags["target_profit"] = None
    flags["stop_loss"] = None
    flags["holding_period"] = None
```

### 2. 점수 구성에 따른 세부 분류

**스윙 트레이딩 (10점 이상):**
- 골든크로스(3) + 거래량(2) + 모멘텀 지표(5점 이상) = 빠른 상승 가능
- 단기 모멘텀에 집중

**포지션 트레이딩 (8~9점):**
- 골든크로스(3) + 추세 지표(5점 이상) = 중기 추세 형성
- 안정적인 상승 추세

**장기 투자 (6~7점):**
- 기본 신호 충족 + 추세 지표 = 장기 성장 가능
- 지속 가능한 성장

## 점수 구성 분석

### 스윙 트레이딩 (10점 이상) 특징

**일반적인 구성:**
- 골든크로스(3) + 거래량(2) + MACD(1) + RSI(1) + TEMA(2) + OBV(2) = 11점
- 또는 골든크로스(3) + 거래량(2) + 여러 모멘텀 지표(5점 이상)

**특징:**
- 거래량 급증이 필수
- 강한 모멘텀 지표
- 빠른 상승 가능

### 포지션 트레이딩 (8~9점) 특징

**일반적인 구성:**
- 골든크로스(3) + 추세 지표(5점 이상)
- 또는 거래량(2) + 여러 추세 지표(6점 이상)

**특징:**
- 안정적인 추세 형성
- 지속 가능한 상승
- 중기 보유 적합

### 장기 투자 (6~7점) 특징

**일반적인 구성:**
- 기본 신호 3~4개 충족
- 추세 지표 일부 충족

**특징:**
- 기본적인 상승 신호
- 안정적인 기업
- 장기 성장 가능

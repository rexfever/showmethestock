# 매매 전략 개선 방안

## 현재 전략의 문제점 분석

### 현재 전략
- **손절**: -5%
- **익절**: +8%
- **보존**: +3%
- **최대 보유**: 60일

### 문제점
1. **익절 종목 0개 (0.0%)**: +8% 익절 기준에 도달하는 종목이 없음
2. **손절 비율 98.6%**: 대부분 손절로 종료
3. **평균 수익률 -3.96%**: 지속적인 손실
4. **평균 보유 기간 1.6일**: 매우 짧은 보유 기간

## 개선 방안

### 1단계: 익절 기준 완화 (즉시 적용)

**현재**: 익절 +8%
**개선**: 익절 +3% ~ +5%

**이유**:
- +8%는 현실적으로 도달하기 어려움
- 성과보고서에서 max_return 5.16% (평균)을 고려하면, +3~5%가 현실적
- 작은 수익을 확실히 실현하는 것이 중요

### 2단계: 손절 기준 완화 (리스크 허용)

**현재**: 손절 -5%
**개선**: 손절 -7% ~ -10%

**이유**:
- -5%는 너무 빨리 손절 발생
- 주식 시장의 변동성을 고려하면 -7~10%가 적절
- 보유 기간을 늘려 회복 기회 제공

### 3단계: 보존 조건 강화 (더 빠른 보존)

**현재**: 보존 +3%
**개선**: 보존 +1.5% ~ +2%

**이유**:
- +3%까지 기다리면 이미 손절될 수 있음
- 더 빠르게 보존 활성화하여 손익분기점 보호
- 작은 수익을 보존하는 것이 중요

### 4단계: 최소 보유 기간 설정 (신규)

**추가**: 최소 보유 3~5일 강제

**이유**:
- 너무 빠른 손절 방지
- 시장 변동성에 대한 여유 제공
- 단기 변동성에 휘둘리지 않도록

## 추천 전략 조합

### 전략 A: 보수적 (안정적 수익 추구)
- **손절**: -7%

- **보존**: +1.5%
- **최소 보유**: 3일

**특징**:
- 작은 수익을 확실히 실현
- 빠른 보존으로 손익분기점 보호
- 적당한 손절 여유 제공

### 전략 B: 균형형 (현실적 균형)
- **손절**: -8%
- **익절**: +4%
- **보존**: +2%
- **최소 보유**: 3일
- **최대 보유**: 40일

**특징**:
- 손절과 익절의 균형
- 보존 조건으로 안정성 확보

### 전략 C: 공격적 (높은 수익 추구)
- **손절**: -10%
- **익절**: +5%
- **보존**: +2.5%
- **최소 보유**: 5일
- **최대 보유**: 60일

**특징**:
- 높은 손절 허용으로 회복 기회 확대
- 높은 익절 목표
- 긴 보유 기간으로 성장 기회 제공

## 전략 검증 스크립트 수정 필요사항

1. **최소 보유 기간 로직 추가**
   ```python
   if current_days < min_hold_days:
       # 최소 보유 기간 전에는 손절 제외
       continue
   ```

2. **트레일링 스톱 추가 검토**
   - 보존 후 최고가 대비 일정 비율 하락 시 매도
   - 예: 보존 후 최고가 대비 -3% 하락 시 매도

3. **부분 매도 로직 검토**
   - 익절 기준 도달 시 50% 매도, 나머지는 트레일링 스톱
   - 리스크 분산 효과
